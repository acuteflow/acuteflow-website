Política de Privacidade

Versão: 1.0
Última atualização: 26 de março de 2026

1. Introdução

Esta Política de Privacidade explica como lidamos com as informações em nossos aplicativos móveis ("Apps"). Nossos aplicativos são projetados para funcionar localmente em seu dispositivo, com uma abordagem que prioriza a privacidade. Ao usar nossos aplicativos, você concorda com os termos descritos aqui.

2. Aplicativos Abrangidos

Esta Política de Privacidade aplica-se aos seguintes itens:

  • Game Turn Timer (Android)
  • Todos os futuros aplicativos publicados sob esta conta de desenvolvedor, a menos que uma política específica seja fornecida.

3. Coleta e Uso de Informações

3.1 Informações que NÃO Coletamos

Não coletamos, armazenamos ou transmitimos nenhum dos seguintes dados diretamente para nossos próprios servidores:

  • Identidade pessoal (nomes, e-mails, números de telefone ou contas).
  • Dados de localização GPS precisos.
  • Conteúdo gerado pelo usuário ou histórico de jogo.

3.2 Provedores de Serviços de Terceiros (Publicidade)

Para manter nossos aplicativos gratuitos, usamos o Google AdMob para exibir anúncios. O AdMob pode coletar e processar dados automaticamente, incluindo:

  • Identificadores de Dispositivo: Como o ID de publicidade do Google (AAID).
  • Localização Geral: Inferida do seu endereço IP (usada para exibir anúncios no idioma correto).
  • Dados Técnicos: Modelo do dispositivo, versão do sistema operacional e registros de desempenho do aplicativo.

Esses dados são processados pelo Google conforme descrito na Política de Privacidade do Google.

4. Conformidade Legal Global

Reconhecemos os direitos de privacidade globalmente, incluindo o RGPD (Europa), UK GDPR (Reino Unido), CCPA (EUA), LGPD (Brasil), PIPA (Coreia do Sul) e APPI (Japão).

  • Consentimento: Para usuários em regiões regulamentadas, usamos uma Plataforma de Gestão de Consentimento (CMP) certificada pelo Google para coletar suas preferências.
  • Opt-Out: Você pode redefinir seu ID de publicidade nas configurações do seu dispositivo Android (Configurações > Google > Anúncios).

5. Retenção e Exclusão de Dados

Não armazenamos dados de usuários em nossos servidores. Quaisquer dados coletados pelo Google AdMob são retidos de acordo com as políticas de retenção de dados do Google. Como não temos acesso aos seus identificadores pessoais, não podemos atender a solicitações individuais de exclusão de dados mantidos por terceiros.

6. Privacidade das Crianças

Nossos aplicativos não são direcionados a crianças menores de 13 anos. Não coletamos intencionalmente dados pessoais de crianças. Se você acredita que uma criança forneceu dados por meio de nosso parceiro publicitário, entre em contato conosco.
